# -nice-
仿nice图片打标签
本项目是基于PictureView进行优化的，github地址：https://github.com/saiwu-bigkoo/Android-PictureTagView 
优化亮点：

1.支持点击标签切换方向。

2.支持长按标签删除。

3.支持长按标签移动。

4.支持获取图片所有标签信息集合，包括标签x、y轴坐标，标签内容，标签id，标签方向，标签类型。

5.支持打不同类型的标签，如推荐标签、位置标签、活动标签等。
